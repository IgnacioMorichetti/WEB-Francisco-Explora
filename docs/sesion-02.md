# Sesión 02 — 2026-06-05

## Resumen

Sesión de definición de stack, diseño y estructura del sitio.

## Decisiones tomadas

### Stack
- **Astro + Tailwind CSS**
- Despliegue en Vercel (gratuito)
- Estructura de una sola página con scroll continuo

### Identidad visual
- Estética: **Explorador terroso** (diario de viaje, cuaderno de explorador)
- Fondo: `#FAF7F2` (crema)
- Acento 1: `#8B5E3C` (marrón)
- Acento 2: `#4A7C59` (verde oliva)
- Títulos: tipografía serif (Playfair Display o Lora)
- Cuerpo: tipografía sans-serif (Inter o DM Sans)

### Secciones (en orden)
1. **Hero** — muy importante, presentación impactante
2. **Sobre mí** — muy importante, quién es Francisco y sus valores
3. **El Desafío (Ironman)** — muy importante, el Medio Ironman 70.3 a finales de junio 2026
4. **Galería** — fotos de aventuras y entrenamientos
5. **Redes + Contacto** — muy importante, links a Instagram y contacto para colaboraciones

### Eje del sitio
Francisco como persona — el Ironman es parte de su historia pero no lo define todo. La personalidad (explorador, humilde, honesto, divertido, sociable) es el centro.

### Bitácora / Blog
No se incluye en esta versión. Idea guardada para una segunda etapa.
